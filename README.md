# Rekova Recovery

A modern, fully responsive marketing website for **Rekova Recovery** — a premium wellness and recovery delivery company.

## About

Rekova Recovery delivers elite-level recovery products (cold therapy, compression therapy, recovery nutrition, heat therapy, mobility kits, and sleep bundles) directly to athletes, gyms, hotels, and homes.

## Project Structure

```
RekovaRecovery/
├── index.html        # Main homepage (single-page layout)
├── css/
│   └── styles.css    # All styles — CSS custom properties, responsive grid, animations
├── js/
│   └── main.js       # Interactions: mobile nav, scroll reveal, form validation
└── README.md
```

## Features

- **Hero section** with animated floating cards and key stats
- **Services grid** — 6 recovery categories with feature lists
- **How It Works** — 3-step process
- **About** section with illustrated stats
- **Testimonials** from real client personas
- **Order / Contact form** with client-side validation and success state
- **Fully responsive** — mobile, tablet, and desktop layouts
- **Smooth scroll**, sticky header, scroll-reveal animations
- **Accessible** — ARIA labels, semantic HTML, keyboard-navigable

## Getting Started

This is a static website. Simply open `index.html` in any modern browser — no build step required.

```bash
# Serve locally (optional)
npx serve .
# or
python3 -m http.server 8080
```
